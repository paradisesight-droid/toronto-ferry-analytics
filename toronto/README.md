# Real-Time Ferry Ticket Sales & Redemption Analytics for Toronto Island Park

## Description
This project analyzes historical ferry ticket sales and redemption data to identify passenger demand patterns and predict future ticket sales using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Streamlit

## Features
- Data Cleaning
- Exploratory Data Analysis
- KPI Dashboard
- Machine Learning Prediction
- Interactive Streamlit Dashboard
